# candle_xps

Random experiments with `candle`.

Reference python implementations can be found in `py` directory.

Experimental rust implementations are in `rs` directory.

## Index

- [py/rfdetr](py/rfdetr) - cut-down fork of apache2 bits of rf-detr for study purposes
- [py/torch_study](py/torch_study) - random pytorch stuff for comparisons with candle
- [rs/candle_conv](rs/candle_conv) - candle conv performance investigations
- [rs/candle_rf_detr](rs/candle_rf_detr) - a working, but dirty port of RF-detr to candle
- [rs/candle_yolov8](rs/candle_yolov8) - copy-pasted yolov8 example from candle-examples for convenient comparisons with rf-detr
- [rs/cocotools](rs/cocotools) - vibe-coded port of pycocotools to rust for running AP evals for rf-detr port
